## 命名规范

### 项目命名

全部采用小写方式， 以下划线分隔。
例：`my_project_name`

### 目录命名

参照项目命名规则；

有复数结构时，要采用复数命名法。

例：`scripts`，`styles`，`images`，`data_models`

vue 的项目中，components 下的组件目录名，使用大驼峰命令

例：`LeftBar`

### JS 文件命名

参照项目命名规则。

例：`account_model.js`

### CSS, SCSS 文件命名

参照项目命名规则。

例：`retina_sprites.css`

### HTML 文件命名

参照项目命名规则。

例：`error_log.html`
